# Chatroom

This project aims to provide a simple environment for users to chat with one another.

## How to use

To run this project properly, you need to install [Node.js](https://nodejs.org/en/download/package-manager/current) and [Yarn package](https://classic.yarnpkg.com/lang/en/docs/install/#debian-stable) first.

Then run the command below in order to install necessary dependencies which can be found in the `package.json` file and followed by that, `node_modules` file will be created.

```bash
yarn install
```

Next step involves around running back-end and front-end servers respectively.
Open the terminal and split it into two separate sections each for server and client.

In the server terminal run the command below:

```bash
yarn server:dev
```

Now your back-end server is running.

And for the client run the command below:

```bash
yarn client:dev
```

Then you can follow the link provided in client terminal to use the project.
